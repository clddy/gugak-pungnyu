# 포디엄 (프로토타입)

클래식 음악인을 위한 플랫폼 프로토타입 — 뮬(Mule)의 클래식 버전을 목표로.

- **구인구직** (`jobs.html`): 객원/대타, 반주, 행사연주, 강사 등 유저 게시판 (목데이터)
- **공식 채용공고** (`official.html`): 전국 교향악단·합창단·오페라단 20개 기관 홈페이지의 단원 모집 공고를 크롤러(`crawler/`)가 매일 자동 수집
- 연습실 / 과외 / 중고거래 / 커뮤니티: 예정

## 크롤러

```
python crawler/main.py
```

기관별 파서(`crawler/sources.py`)가 각 악단 게시판을 읽고, PDF/HWP 첨부에서 접수 마감일을 추출해(`crawler/attach.py`) `data/official-data.js`로 내보냅니다.

피드백 환영: 우측 하단 💬 버튼
